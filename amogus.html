Latihan 1: Styling Teks

Buatlah paragraf dengan ketentuan:

  • Warna teks: #2c3e50
  • Ukuran font: 18px
  • Tebal font: 500
  • Jarak antar baris: 1.6
  • Rata kiri-kanan (justify)

Ini adalah teks contoh. Ubah style teks ini sesuai dengan instruksi di atas.

CSS yang digunakan:

p {
  color: #2c3e50;
  font-size: 18px;
  font-weight: 500;
  line-height: 1.6;
  text-align: justify;
}

Latihan 2: Box Model

Buatlah sebuah div dengan ketentuan:

  • Lebar: 250px, Tinggi: 150px
  • Padding: 20px
  • Border: 3px solid #3498db
  • Margin: 15px
  • Background color: #ecf0f1
  • Border radius: 8px
Ini adalah kotak. Style div ini sesuai dengan instruksi di atas.

CSS yang digunakan:

div {

  padding: 20px;
  border: 3px solid #3498db;
  margin: 15px;
  background-color: #ecf0f1;
  border-radius: 8px;
}

Latihan 4: Hover Effects

Buatlah sebuah tombol dengan efek hover dengan ketentuan:

  • Tombol: Padding 12px 24px, Background #2ecc71, Warna teks putih, Border none, Border radius 4px, Font size 16px
  • Efek hover: Background #27ae60, Transform scale(1.05), Transition 0.3s
  • Efek aktif: Transform scale(0.95)

CSS yang digunakan:

.custom-btn {
  padding: 12px 24px;
  background-color: #2ecc71;
  color: white;
  border: none;
  border-radius: 4px;
  font-size: 16px;
  cursor: pointer;
  transition: all 0.3s ease;
}

.custom-btn:hover {
  background-color: #27ae60;
  transform: scale(1.05);
}

.custom-btn:active {
  transform: scale(0.95);
}

Latihan 5: Card Component

Buatlah komponen kartu (card) dengan ketentuan:

  • Kartu: Lebar 300px, Border radius 10px, Overflow hidden, Box shadow, Background putih
  • Gambar: Lebar 100%, Tinggi 180px, Object-fit cover
  • Konten: Padding 20px
  • Judul: Font size 20px, Margin bottom 10px
  • Tombol: Background #3498db, Warna putih, Padding 8px 16px, Border radius 4px, Border none, Display inline-block
  • Efek hover pada tombol: Background #2980b9

Judul Kartu

Ini adalah deskripsi kartu. Ini adalah contoh pembuatan komponen kartu dengan HTML dan CSS.

CSS yang digunakan:

.card {
  width: 300px;
  border-radius: 10px;
  overflow: hidden;
  box-shadow: 0 4px 8px rgba(0, 0, 0, 0.1);
  background-color: white;
}

.card img {
  width: 100%;
  height: 180px;
  object-fit: cover;
}

.card-content {
  padding: 20px;
}

.card-content h3 {
  font-size: 20px;
  margin-bottom: 10px;
  color: #2c3e50;
}

.card-content p {
  color: #7f8c8d;
  margin-bottom: 15px;
}

.card-content button {
  background-color: #3498db;
  color: white;
  padding: 8px 16px;
  border-radius: 4px;
  border: none;
  cursor: pointer;
  transition: background-color 0.3s;
}

.card-content button:hover {
  background-color: #2980b9;
}

Latihan 6: Responsive Navigation

Buatlah navigasi responsif dengan ketentuan:

  • Navbar: Background #2c3e50, Padding 15px, Display flex, Justify-content space-between, Align-items center
  • Logo: Warna putih, Font size 20px, Font weight bold
  • Menu: Display flex, List-style none
  • Item menu: Margin 0 15px
  • Link: Warna putih, Text-decoration none, Transition 0.3s
  • Efek hover pada link: Warna #3498db
  • Tombol toggle: Display none secara default (untuk mobile)
Logo
  • Beranda
  • Tentang
  • Layanan
  • Kontak
☰

CSS yang digunakan:

.navbar {

}

}

.nav-menu {

}
